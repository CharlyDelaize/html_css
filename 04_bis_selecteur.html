Aide mémoire des sélécteurs CSS déjà vus

  • * sélectionne TOUS les éléments de la page HTML.
  • .ma-jolie-nav séléctionne tous les éléments ayant le nom de class "ma-jolie-nav".
  • #mon-unique-id séléctionne l'élément ayant le nom d'id "mon-unique-id".
  • div, article, aside séléctionne toutes les balises <div> <article> et <aside>.
  • nav a séléctionne tous les liens <a> inclus /imbriqués dans la balise <nav>.
  • nav a:hover séléctionne tous les liens de la nav et active une animation lors du survol de la souris.

Propriété CSS utilisées dans cette page

Décrivez leur usage et leurs valeurs possibles

  1. background :permet de définir les différentes valeurs des propriétés liés à la gestion d'un éléments
  2. color : L'attribut color est utilisée pour définir une couleur (couleur, rgb, #000000)
  3. float : retire un élément du flux, pour le placer à gauche ou à droite de son conteneur
  4. font-family : changer la police d'écriture (sans-serif, fantasy, cursive...)
  5. font-size : changer la taille du texte (px, rem, em...)
  6. height : changer la hauteur (% em...)
  7. left : positionner au bord gauche de l'écran, en px ou autres
  8. line-height : changer la hauteur entre les lignes
  9. margin : change la taille des marges extérieures
  10. padding : change la taille des marges intérieures
  11. position : définit la position d'un élément sur la page (fixe, absolute...), s'utilise avec left, top, right, bottom
  12. text-align : permet d'aligner le texte (left, right, center, justify)
  13. text-decoration : dans notre cas, on l'a utilisé pour retirer l'underline des a avec la valeur none
  14. top : positionner en haut de l'écran, en px ou autres
  15. transition : determine la durée de l'animation s'il y a un ":hover" (s) (property, duration, timing-function, delay)
  16. width : défini la largeur d'un élément (%)

Propriété CSS à rechercher

rechercher les Propriétés ci-dessous, faites des tests sur cette page en utilisant la console de débogage pour tester les valeurs qu'elles peuvent prendre (F12 dans le navigateur)

  1. font-weigth : définit la graisse de la police (normal, bold, lighter ou bolder, on peut meme utiliser des valeurs de 1 à 1000 même avec des virgules)
  2. letter-spacing : définit l'espace entre les lettres (normal, em, px, voire des virgules)
  3. word-spacing : définit l'espace entre les mots (normal, em, px, voire des virgules)
  4. text-shadow : met de l'ombre au texte (axe x y fondu, couleur (1px 1px 2px black))
  5. background-image : met un motif en arrière plan, on peut mettre un dégradé ou une url, on peut superpositionner plusieurs choses (url("css/montagnes.png"))
  6. background-size : modifie la taille du fond, (cover, contain ou des valeurs)
  7. opacity : determine la transparence de 0 (invisible) à 1 visible en passant par 0.5 à moitié visible.
  8. border-style : détermine le type de bordure d'un tableau ou d'un block (dotted, dashed, solid, double, groove, ridge) fonctionne comme pour le padding: top right bottom left
  9. box-shadow : met de l'ombre à une balise (x | y | blur-radius | spead-radius, | color, ewemple: 2px 2px 2px 1px rgba(0, 0, 0, 0.2))
  10. list-style-image : met une image à la place d'un point dans une liste à puces: url("css/montagnes.png")
  11. border : créer une bordure, on peut y mettre plusieurs valeurs (largeur, type de trait, couleur)
  12. border-radius : fait des bords arrondis (haut à gauche | haut à droite | bas à droite | bas à gauche
  13. border-width : met une epaisseur à la bordure (px, em, mais aussi : thin, medium, thick)
  14. font-variant : modifie la police en fonction de la valeur (font-variant : small-caps)
  15. background-color : change la couleur d'arrière-plan de la balise, accepte (rgb, hexadecimaux, nom de valeur en anglais, hsl hsla (comme rgb mais avec %)
  16. !important : à mettre à la fin d'une ligne de CSS pour forcer le navigateur à appliquer le style, il ne sera pas surchargé et il faut éviter de l'utiliser
